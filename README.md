# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer that analyzes resumes, provides an ATS score, identifies strengths and weaknesses, suggests improvements, and recommends suitable job roles using Artificial Intelligence.

The application allows users to upload a resume PDF and receive an AI-generated analysis report to improve their chances of passing Applicant Tracking Systems (ATS).

---

## 🚀 Features

- 📄 Upload resume in PDF format
- 🔍 Extract text from resume automatically
- 🤖 AI-powered resume analysis
- 📊 ATS compatibility score
- ✅ Identify resume strengths
- ⚠️ Find weaknesses and improvement areas
- 🎯 Suggest missing skills
- 💼 Recommend suitable job roles
- 🌐 Full-stack web application
